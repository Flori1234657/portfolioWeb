# Portfolio Website 🌐

[Live Demo](https://portfolio-project-45af5.web.app/)

Simple portfolio website showcasing some of my projects, skills, and experience in web development.

## Features ✨
- Responsive design for all screen sizes
- Project showcase with 3 images
- Some of my skills
- Contact inquiry with email integration
- Smooth animations and transitions
- Performance optimization

## Technologies Used 🛠️
**Frontend:**
- React.js with Vite
- Sass
- JavaScript (ES6+)
- Framer Motion

**Hosting & Services:**
- Firebase

## Installation 💻
1. Clone the repository:
```bash
git clone https://github.com/Flori1234657/portfolioWeb.git
```
2. Install dependencies:
```bash
yarn
```
3. Start development server:
```bash
yarn dev
```
